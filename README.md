# RoadSoS – AI-Powered Emergency Response Platform

RoadSoS is a project developed for the IIT Madras Road Safety Hackathon 2026.  
The goal is to provide quick, reliable access to emergency services during road accidents, helping reduce delays in the critical golden hour.

---

## Overview
RoadSoS is a simple, location-based platform that connects users to nearby trauma centers, ambulance services, police stations, and towing facilities.  
It combines maps, emergency contacts, and a chatbot interface into one easy-to-use dashboard.

---

## Features
- Emergency Map showing nearest hospitals, police stations, and towing services  
- Dashboard cards for services with urgency indicators  
- Chatbot assistant for quick queries like “nearest ambulance”  
- Responsive design for desktop and mobile  
- Offline fallback with cached emergency contacts  

---

## Tech Stack
- **Frontend:** React / Next.js  
- **Backend:** Node.js + Express or Python + FastAPI  
- **APIs:** Google Maps, OpenStreetMap  
- **AI:** Chatbot powered by AI models
